Informacje o projekcie

Projekt symuluje działanie koszyk przy pomocy wykorzystania bazy Riak. Z powodu, że nie ma możliwośći instalacji Riaka na systemie operacyjnym Windows, przy tworzeniu skorzystałem z Dockera - "DevDbRiak". Oprócz angular.js musiałem użyć jeszcze node.js, poniewać występował problem z CORS. W tym wypadku node.js służy tylko i wyłącznie do serwowania plików oraz jako proxy do HTTP API Riak.
Za pomocą nazwy użytkownika (widocznej na górze) może sobie symulować sesję koszyka - inna nazwa użytkownika = inny koszyk.
